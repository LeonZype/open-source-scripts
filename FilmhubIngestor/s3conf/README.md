The following configurations must be set in order to the AWS-CLI S3 function.

- **AWS_ACCESS_KEY_ID**

- **AWS_SECRET_ACCESS_KEY**

- **AWS_DEFAULT_REGION**

- **AWS_DEFAULT_OUTPUT**
